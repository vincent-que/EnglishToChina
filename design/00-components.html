17 · 折叠面板 Accordion

翻译风格说明
学术正式：适合论文、研究报告等学术场景，用词严谨，句式规范。
商务通用：适合商务邮件、报告、合同等，专业但不生硬。
日常口语：适合日常文档、博客等，通俗易懂，自然流畅。
支持的文件格式
当前支持 PDF（文字版 + 扫描版）和 Word (.docx/.doc) 格式。后续版本将支持 PPT、Excel 等格式。
API 额度如何计算？
API 额度按翻译消耗的 token 数量计费。平均一页 A4 文档约消耗 2000-3000 token。建议使用 DeepSeek 模型，性价比最高。
