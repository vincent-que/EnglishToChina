# 商业模式与运营设计

## 1. 定价模型

| 版本 | 价格 | 功能范围 | 内置 API 额度 | 授权方式 |
| --- | --- | --- | --- | --- |
| 基础版 | ¥29-49 | Word 翻译、基础表格 | 50 页/月 | 单人授权 |
| 专业版 | ¥69-99 | PDF、Word、复杂表格、OCR | 200 页/月 | 单人授权 |
| 终身版 | ¥129-199 | 全部功能、优先更新 | 自带 Key 或不限额方案 | 单人授权 |
| 企业版 | ¥499+/年 | 私有部署、团队术语库、批量授权 | 自定义 | 团队授权 |

## 2. 成本结构

| 成本项 | 说明 | 估算 |
| --- | --- | --- |
| 大模型 API | DeepSeek、通义千问、Kimi 等 | 按页数和 token 消耗计费 |
| 授权服务 | Serverless API + 数据库 | 小规模可低成本运行 |
| 下载分发 | 安装包、更新包 CDN | 按流量计费 |
| 客服 | 激活、排版、API Key 配置答疑 | 按用户规模增长 |

## 3. 运营重点

- 先保证 PDF/Word 转换质量和中文覆盖率，这是付费转化的核心。
- 默认使用用户自己的 API Key，降低运营方垫付模型费用的风险。
- 内置 API 额度只作为增值能力，不作为基础运行依赖。
- 授权系统采用隐私优先方案，不采集硬件标识，也不做本机绑定。
- 售后文档重点覆盖 API Key 配置、连接测试、排版异常处理、术语表导入。

## 4. 推荐销售路径

1. 免费试用允许少量页数转换，让用户验证排版和翻译质量。
2. 专业版提供批量转换、PDF 输出、术语表和历史记录。
3. 企业版提供团队术语库、私有部署、批量授权和定制排版适配。

## 5. 常见问题

| 问题 | 建议回复 |
| --- | --- |
| 如何获取 API Key？ | 在设置页选择引擎后，前往对应平台创建 API Key，并使用“测试连接”确认可用。 |
| 翻译后排版乱了怎么办？ | 优先使用文字版 PDF 或 Word 源文件；复杂 PDF 建议先输出 Word 检查，再导出 PDF。 |
| 授权码无法激活？ | 检查网络连接、授权码是否输入完整，以及授权码是否过期或被撤销。 |
| 如何在新电脑上使用？ | 安装程序后重新输入授权码即可，不需要解绑旧电脑。 |
| 翻译速度很慢？ | 检查网络，尝试切换翻译引擎，或降低并发任务数量。 |
| 支持哪些文档格式？ | 当前重点支持 PDF 和 Word，后续可扩展 Excel、PPT。 |
